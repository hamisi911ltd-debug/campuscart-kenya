# Quick Checkout Guide - Location-Based System

## 🛒 How to Place an Order (New System)

### Step 1: Add to Cart & Checkout
1. Browse products
2. Click product → Select quantity
3. Click **"Checkout"** button (green)
4. Review cart items

---

### Step 2: Enter Delivery Address
```
┌─────────────────────────────────────┐
│ Delivery Address                    │
│                                      │
│ [Hostel Block A, Room 205]          │
│                                      │
│ Provide detailed delivery           │
│ instructions for the rider          │
└─────────────────────────────────────┘
```

**Examples of Good Addresses**:
- ✅ "Hostel Block A, Room 205, near main gate"
- ✅ "Jomo Kenyatta Hall, 3rd Floor, Room 3B"
- ✅ "Behind library, blue building, Room 12"

**Examples of Bad Addresses**:
- ❌ "Hostel" (too vague)
- ❌ "Room 205" (which block?)
- ❌ "Campus" (which location?)

---

### Step 3: Share Your Location 📍

#### First Time:
```
┌─────────────────────────────────────┐
│ 📍 Your Location                    │
│                                      │
│ Share your live location for        │
│ accurate delivery                    │
│                                      │
│ [📍 Share My Location]              │
└─────────────────────────────────────┘
```

1. Click **"Share My Location"** button
2. Browser will ask for permission
3. Click **"Allow"** or **"Allow Once"**
4. Wait for location to be captured

#### Browser Permission Popup:
```
┌─────────────────────────────────────┐

│ Know your location                  │
│                                      │
│ [Block]  [Allow Once]  [Allow]      │
└─────────────────────────────────────┘
```

**Always click "Allow" or "Allow Once"**

## 🗺️ Location Features

### Why Share Location?
- ✅ **Accurate Delivery**: Rider knows exact location
- ✅ **No Confusion**: No need to call for directions
- ✅ **Faster Service**: Direct navigation to you
- ✅ **Better Experience**: Smooth delivery process

### What Location Shows:
- Your exact GPS coordinates
- Clickable Google Maps link
- Street-level accuracy
- Real-time position

### Privacy:
- ✅ Location shared only with admin
- ✅ Used only for this delivery
- ✅ Not stored permanently
- ✅ You control when to share

## ❌ Troubleshooting

### Problem: "Location permission denied"

**Solution**:
1. Click lock icon in browser address bar
2. Find "Location" permission
3. Change to "Allow"
4. Refresh page
5. Try "Share My Location" again

**On Mobile**:
1. Go to Settings
2. Find your browser (Chrome/Safari)
3. Enable Location Services
4. Return to checkout
5. Try again

---

### Problem: "Location information unavailable"

**Solution**:
1. Check if GPS is enabled on device
2. Move to area with better signal
3. Try outdoors if indoors
4. Wait a few seconds
5. Click "Share My Location" again

---

### Problem: Button says "Share Location to Continue"

**Reason**: Location not shared yet

**Solution**:
1. Scroll up to "Your Location" section
2. Click "Share My Location" button
3. Allow browser permission
4. Wait for map to appear
5. Button will change to "Complete Order Placement"

---

### Problem: Map not showing

**Solution**:
1. Check internet connection
2. Refresh the page
3. Clear browser cache
4. Try different browser
5. Contact admin if issue persists
