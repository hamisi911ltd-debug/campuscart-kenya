# 📱 How to Install CampusMart App

## For iPhone/iPad Users (iOS)

1. **Open Safari** (must use Safari browser)
2. Visit the CampusMart website
3. Tap the **Share button** (⎙) at the bottom of the screen
4. Scroll down and tap **"Add to Home Screen"**
5. Tap **"Add"** in the top right corner
6. Done! The app icon appears on your home screen

### iOS Tips:
- ✅ Must use Safari (not Chrome or other browsers)
- ✅ App works offline after installation
- ✅ Looks and feels like a native app
- ✅ No App Store needed

---

## For Android Users

### Method 1: Auto Prompt (Easiest)
1. Open the website in Chrome
2. Wait 5 seconds
3. Tap **"Install"** on the popup
4. Done!

### Method 2: Manual Install
1. Open the website in Chrome
2. Tap the **menu** (⋮) in the top right
3. Tap **"Install app"** or **"Add to Home screen"**
4. Tap **"Install"**
5. Done! Find the app in your app drawer

### Android Tips:
- ✅ Works in Chrome, Edge, Samsung Internet
- ✅ Full offline support
- ✅ Receives notifications
- ✅ Updates automatically

---

## For Desktop Users (Windows/Mac)

### Chrome/Edge:
1. Visit the website
2. Look for the **install icon** (⊕) in the address bar
3. Click it and select **"Install"**
4. App opens in its own window

### Alternative:
1. Click the **menu** (⋮) in the top right
2. Select **"Install CampusMart..."**
3. Click **"Install"**

## Troubleshooting

**"Install" option not showing?**
- iOS: Make sure you're using Safari
- Android: Make sure you're using Chrome
- Desktop: Try Chrome or Edge browser

**App not working offline?**
- Open the app while online first
- Browse a few pages to cache content
- Then try offline mode

**Want to uninstall?**
- iOS: Long-press the app icon → Remove App
- Android: Long-press → Uninstall
- Desktop: Right-click app icon → Uninstall
